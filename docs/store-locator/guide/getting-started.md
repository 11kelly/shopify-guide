# 快速开始

欢迎使用 **Jika Store Locator**！本指南将帮助您在几分钟内完成设置，并在您的 Shopify 店铺中上线门店定位器。

## 1. 安装应用

从 Shopify 应用商店安装应用后，您将进入应用仪表盘（Home）。
![应用仪表盘](/store-locator/images/dashboard-home.png)

## 2. 配置 Google Maps API

门店定位器需要 Google Maps API 密钥来渲染地图和解析地址。

1. 点击Home页中的**Check API key**或进入应用设置中的 **Integrations (集成)** 标签页。
![Google Maps 配置](/store-locator/images/google-maps-settings.png)

2. 输入您的 **Google Maps API Key**。 
   - *注意：请确保已在 Google Cloud 控制台中启用了 Maps JavaScript API ,Places API和 Geocoding API。*
3. 点击 **Save (保存)**。

## 3. 添加门店 (Add Store)

1. 导航到应用首页或菜单中的 **Stores (门店)** 部分并点击 **Add Store (添加门店)** 或 **Add First Store**。您也可以选择 **Import CSV** 批量导入。
![门店列表页面](/store-locator/images/stores-index.png)
2. 填写门店名称、电话、网站等并上传门店图片（可选）。
![添加门店表单](/store-locator/images/add-store-form.png)

3. 或点击右上角**pick from map**使用在谷歌地图上找到您店铺的位置，自动带入相关信息。
![从地图选取位置](/store-locator/images/pick-from-map.png)

4. 将门店状态设为 **Active (启用)** 并点击 **Save (保存)**。

## 4. 在前台发布定位器

我们提供了 3 种灵活的发布模式（独立页面、主题区块、悬浮挂件）。

1. 回到首页，在 **Quick Setup Roadmap** 中找到第三步 **Publish Store Locator**，点击 **Publish Options**。
2. 在 Publish 页面，您可以选择：
   - **Mode 1: Managed Storefront Page**（自动创建独立页面，支持 Classic 和 Stacked 布局）。
   - **Mode 2: Theme Section**（在您现有的页面中插入地图区块，支持全量 9 种布局）。
   - **Mode 3: Floating Widget**（全局悬浮地图按钮）。
   ![添加地图](/store-locator/images/publish-store-locator.png)
3.如果您选择 Mode 1，直接点击Create storefront Page创建页面，随后点击View Live Page,即可快速创建前台地图。
![添加地图](/store-locator/images/mode-map-1.png)
![添加地图](/store-locator/images/mode-map-2.png)
4. 如果您选择 Mode 2 添加到现有页面，点击 **Open Theme Editor**，在左侧侧边栏点击 **Add section**，在 Apps 中选择 **Jika Store Locator** 即可。
![添加地图](/store-locator/images/add-section.png)
5. 您可以前往首页点击 **APPEARANCE SETTINGS** 下的 **Manage Appearance** 来自由配置地图颜色、标记和卡片圆角等外观样式。
6. 点击 **Save (保存)** 即可上线。

现在，您的客户就可以在前端查找您的门店了！
