# 门店管理

在 Jika Store Locator 控制面板中高效管理您的实体门店。

## 添加单家门店

1. 导航到 **Stores (门店)** 页面，点击 **Add Store (添加门店)**。
2. **基本详情**：提供门店名称、电话号码和网址。
3. **地址与坐标**：手动填写或点击右上角**pick from map**使用在谷歌地图上找到您店铺的位置，自动带入相关信息。
![pick from map](/store-locator/images/pick-from-map-1.png)
4. **图片**：上传一张门店图片，它会显示在列表和详情视图中。
5. **标签 (Tags)**：分配标签（如 "24/7", "支持路边自提"）来帮助客户筛选门店。
6. **自定义字段**：添加任何额外数据，如 "店长姓名" 或 "特色服务",字段设置在**Settings > Store Attributes**。

## 门店属性 (Store Attributes)

通过 **标签** 和 **自定义字段**，您可以让门店信息更加丰富，并为客户提供更精准的筛选体验。

### 1. 标签 (Tags)

标签用于对门店进行分类（如：支持维修、24小时营业、有停车位等）。
![tags](/store-locator/images/new-tag.png)

- **全局管理**：进入 **Settings > Store Attributes**。在这里您可以创建、修改或删除全局标签。
- **分配标签**：在添加或编辑门店时，从右侧边栏的 **Tags** 下拉框中选择现有标签，或直接输入新标签并点击 **Create**。
- **前端作用**：开启标签功能后，前台地图会自动显示标签筛选列表，方便用户快速找到符合条件的门店。

### 2. 自定义字段 (Custom Fields)

如果系统自带的字段（名称、地址、电话等）不够用，您可以自定义额外的字段（如：店长姓名、商场楼层、店内促销信息等）。
![fields](/store-locator/images/new-field.png)

- **定义字段类型**：进入 **Settings > Store Attributes** 的 **Custom Field** 区块。
- **添加定义**：点击 **Add Field**，输入字段名称（如 "楼层"）。
- **填写数据**：定义好之后，在添加或编辑门店时，右侧边栏会自动出现这些字段，供您为每家店填写具体内容。
- **前端显示**：填写了内容的自定义字段会自动展示在前台商店详情的列表下方。

## 批量导入门店

如果您有大量门店，可以使用 CSV 文件进行批量导入。

1. 导航到 **Stores > Import**。
![批量导入](/store-locator/images/import-store.png)
2. 下载我们的 **CSV 模板** 以确保数据格式正确。
3. 上传您填好的 CSV 文件。
4. 预览数据并点击 **Start Import (开始导入)**。

## 门店状态

- **Active (启用)**：门店对前端客户可见。
- **Inactive (禁用)**：门店在前端隐藏，但仍保留在后台供您管理。

## 门店类型

- **Offline (线下店)**：标准的实体门店。
- **Online (线上/展示店)**：纯在线存在或虚拟办公室（可根据需要配置筛选）。
