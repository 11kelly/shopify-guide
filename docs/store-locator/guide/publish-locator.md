# 发布定位器 (Publish Store Locator)

在 Jika Store Locator 中，您可以根据店铺的实际需求，灵活选择 3 种不同的发布模式。所有的发布选项都可以在应用后台的 **Publish Options** 中找到。
Mode 1和 Mode 3都需要在主题编辑器中打开App embeds (应用嵌入)。
![悬浮组建](/store-locator/images/jika-store-locator-widget.png)

## Mode 1: Managed Storefront Page (独立页面)

如果您想要一个快速、自动生成的专属地图页面，这是最简单的选择。
注意：Mode 1的页面url不可以修改，如果想自定义url，请选择先在**Online store>Pages**中创建页面，然后使用Mode 2方式注入。
![创建页面](/store-locator/images/managed-storefront-page.png)

1. 进入后台首页，点击 **Publish Options** 或左侧菜单的 **Publish Store Locator**。
2. 填写 **Page Title (页面标题)**。
3. 从下方的 **Select Layout Style** 中选择一种页面布局（支持 Classic 经典左右分栏 和 Stacked 堆叠布局）。
4. 点击 **Create Storefront Page (创建店面页面)**。
5. 系统会自动在您的 Shopify 在线商店中创建一个新的独立页面。您可以通过点击 **View Live** 随时查看效果。

## Mode 2: Theme Section (主题区块)

如果您可以将地图随意嵌入任何页面，或者希望使用更丰富的布局引擎，请选择此模式。
![添加地图](/store-locator/images/add-section.png)

1. 在 Publish 页面，找到 **Mode 2: Theme Section**。
2. 点击 **Open Theme Editor** 按钮，跳转到 Shopify 主题编辑器。 
3. 在目标页面的左侧侧边栏中点击 **Add section (添加区块)**。
4. 在 **Apps (应用)** 列表中，选择 **Jika Store Locator**。
5. 此模式支持全量 **9 种布局风格**（包括 Floating、Modern、Minimal 等），您可以直接在主题编辑器的右侧面板中进行切换和配置。
6. 点击右上角的 **Save (保存)**。

## Mode 3: Floating Widget (悬浮挂件)

悬浮小组件是一个强大的功能，它会在您店铺的每个页面角落放置一个悬浮按钮。点击后，会在优雅的弹窗中打开门店定位器，让客户无需离开当前页面即可查找门店。
![悬浮组建](/store-locator/images/floating-widget.png)

**开启挂件需要双重验证：**

1. 在 Publish 页面的 Mode 3 下，点击 **Configure Widget**，跳转到 **Settings > Widget Settings**。
2. 在 Widget Activation 卡片中，确保勾选了 **Enable Floating Widget (启用悬浮挂件)** 开关。
5. 点击 **Save (保存)**后去前台页面查看悬浮按钮出现没有即可。

挂件的按钮位置、颜色和弹窗布局都可以随时在应用的 **Widget Settings** 中进行调整。
