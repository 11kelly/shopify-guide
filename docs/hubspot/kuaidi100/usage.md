# User Guide

Once configured, you can use the tracking features directly on HubSpot Deal or Order pages.

## 1. Add Tracking Number

Locate the **Kuaidi100 Shipping Tracker** card in the right sidebar of a CRM record page.

1.  Enter the tracking number.
2.  (Optional) For carriers like SF Express that require phone number verification, enter the last four digits of the recipient's phone number.
3.  Click **Add**.

![Add Number](/hubspot/kuaidi100/images/add-shippment-number.png)

## 2. Track Shipment

Once the number is added, the card will show a pending tracking status.

1.  Click the **Track** button.
2.  The system will immediately retrieve real-time logistics information via the API.

<img src="/hubspot/kuaidi100/images/add-shippment-number.png" style="width: 80%; margin: 20px 0; border: 1px solid var(--vp-c-divider); border-radius: 8px;" alt="Pending Status" />

Upon a successful query, the card will display the latest logistics status, carrier information, and delivery updates:

<img src="/hubspot/kuaidi100/images/card.png" style="width: 80%; margin: 20px 0; border: 1px solid var(--vp-c-divider); border-radius: 8px;" alt="Tracking Result" />
